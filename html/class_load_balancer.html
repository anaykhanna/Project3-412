Detailed Description

Simulates a dynamic load balancer.

The LoadBalancer distributes incoming requests across multiple web servers. It automatically scales the number of servers to maintain the queue size between:

  • 50 × number_of_servers (scale down threshold)
  • 80 × number_of_servers (scale up threshold)

It also tracks runtime statistics and applies firewall filtering.
